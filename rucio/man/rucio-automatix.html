Daemon rucio-automatix¶

Automatix is a daemon used to inject random generated files to an RSE. It is used to continuosly check that an RSE is reachable and operating as spected.

usage: rucio-automatix [-h] [--run-once] [--input-file INPUT_FILE] [--threads-per-process THREADS_PER_PROCESS]

Named Arguments¶

--run-once

Runs one loop iteration

Default: False

--input-file

Automatix configuration

Default: ” /opt/rucio/etc/automatix.json”

--threads-per-process

Total number of workers per process

Default: 1
