Daemon rucio-minos-temporary-expiration¶

This special type of minos daemon lists the expired TEMPORARY_UNAVAILABLE replicas and puts them back into AVAILABLE state.

usage: rucio-minos-temporary-expiration [-h] [--run-once] [--threads THREADS] [--bulk BULK] [--sleep-time SLEEP_TIME]

Named Arguments¶

--run-once

Runs one loop iteration

Default: False

--threads

Concurrency control: number of threads

Default: 1

--bulk

Bulk control: number of requests per cycle

Default: 1000

--sleep-time

Seconds to sleep if few requests

Default: 60
