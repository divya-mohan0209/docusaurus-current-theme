Daemon rucio-conveyor-finisher¶

Conveyor is a group of daemons to manage file transfers. The conveyor-finisher is the resposible to update Rucio internal state after the transfer has finished.

usage: rucio-conveyor-finisher [-h] [--run-once] [--total-threads TOTAL_THREADS] [--db-bulk DB_BULK] [--bulk BULK] [--sleep-time SLEEP_TIME] [--activities ACTIVITIES [ACTIVITIES ...]]

Named Arguments¶

--run-once

One iteration only

Default: False

--total-threads

Concurrency control: total number of threads per process

Default: 1

--db-bulk

Bulk control: number of transfers per db query

Default: 1000

--bulk

Bulk control: number of requests

Default: 100

--sleep-time

Seconds to sleep if few requests

Default: 30

--activities

Explicit list of activities to include
