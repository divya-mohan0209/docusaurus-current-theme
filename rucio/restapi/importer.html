POST /import/¶

Import data into Rucio.

Example request:

POST /import HTTP/1.1
Host: rucio.cern.ch

{
    "rses": [{"rse": "MOCK", "rse_type": "TAPE"}]
}

Example response:

HTTP/1.1 201 OK
Vary: Accept
Content-Type: application/x-json-stream

Created
Response Headers

  • Content-Type – application/json

Status Codes

  • 200 OK – DIDs found

  • 401 Unauthorized – Invalid Auth Token

Returns

dictionary with rucio data
